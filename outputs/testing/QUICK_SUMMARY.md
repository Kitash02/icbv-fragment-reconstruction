## What Happened

### The Test
- Compared 1 British Museum fragment against 26 Wikimedia pottery sherds
- All pairs are from DIFFERENT sources (should be rejected)
- Expected: 0 matches
- Actual: 26 matches (25 MATCH, 1 WEAK_MATCH)

### Root Cause
1. **High Color Similarity:** BC = 0.73-0.96 (mean 0.856)
   - Both sources are brownish pottery
   - Color penalty too weak at current settings

2. **Insufficient Penalty:**
   ```
   Current: score_final = geom * (1 - 0.80 * (1 - BC))
   With BC=0.86: penalty = 11.2%
   With geom=0.70: final = 0.622 > 0.55 threshold → MATCH ✗
   ```

3. **Threshold Too Low:**
   - MATCH threshold = 0.55 is too permissive
   - Allows borderline cases through

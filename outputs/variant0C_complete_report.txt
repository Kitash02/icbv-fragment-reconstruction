        if bc_color < 0.75 or bc_texture < 0.70:
            ensemble_verdict = 'WEAK_MATCH'  # Downgrade if appearance marginal

Logic:
- Ensemble voting uses 5 discriminators (geometry, color, texture, gabor, morphology)
- If ensemble suggests MATCH, verify appearance quality
- Reject upgrade if color < 0.75 OR texture < 0.70
- This catches cross-source pairs (good geometry, weak appearance)

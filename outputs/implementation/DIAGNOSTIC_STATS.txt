WHY FEATURES FAIL
================================================================================

1. COLOR SIMILARITY:
   - Different pottery often has similar color palettes
   - Earth tones, terracotta, browns are common
   - BC scores of 0.75-0.85 typical for cross-pottery pairs
   - Threshold of 0.62 is too low

2. CHAIN CODE LIMITATIONS:
   - 8-directional Freeman codes have limited vocabulary
   - Pottery fragments share curved geometry (vessels are round)
   - Break patterns are similar across artifacts
   - Edit distance is forgiving (normalized)

3. TEXTURE COARSENESS:
   - LBP captures coarse texture only
   - Different pottery can have similar surface patterns
   - Need finer-scale texture analysis

4. BONUSES ADD NOISE:
   - Good continuation rewards smooth curves (common in pottery)
   - Fourier captures low-frequency shape (similar across pottery)
   - Both amplify marginal pairs over threshold

================================================================================

ROOT CAUSE: FUNDAMENTAL FEATURE WEAKNESS
================================================================================

The current feature set CANNOT reliably discriminate between:
  - Fragment from pottery A
  - Fragment from pottery B

Because:
  1. Color palettes overlap heavily (earth tones)
  2. Geometric shapes are similar (curved, circular breaks)
  3. Texture features too coarse (LBP insufficient)
  4. No material-specific features (glaze, clay type, firing marks)

Result: Compatibility scores for cross-pottery pairs (0.65-0.80) 
        are too close to within-pottery pairs (0.75-0.90)

Only 0.10-0.15 separation is insufficient for reliable classification
